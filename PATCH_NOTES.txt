星域フロンティア patch v19

目的:
- 「新しい銀河を開始」を押しても始まらない不具合の修正。

原因:
- createGame() の中で buildCluster() 実行中、まだ state が作られていない段階で system(pid) を呼んでいた。
- system() は state.systems を参照するため、ゲーム開始時に例外が出て停止する。

修正:
- buildCluster() の衛星生成処理内だけ、system(pid) ではなく生成中配列 systems[pid] を参照するよう変更。

確認:
- node --check js/game.js: OK
- 疑似DOMスモーク: load / start / galaxy connected / select tutorial moon / explore / endTurn 呼び出しまで OK
- 画像・assets は変更なし。
