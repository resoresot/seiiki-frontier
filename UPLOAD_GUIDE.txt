GitHub Pages へのアップロード手順

1. このZIPを展開します。
2. seiiki-frontier フォルダの中身を GitHub リポジトリ直下にアップロードします。
   - index.html がリポジトリ直下にある状態にしてください。
3. GitHub の Settings → Pages を開きます。
4. Source を「Deploy from a branch」にします。
5. Branch を main、Folder を /root にします。
6. Save して数十秒〜数分待つと公開されます。

操作メモ
- 資源アイコンをクリックすると説明が出ます。
- APバッジをクリックするとAPの説明が出ます。
- 星域マップはドラッグで移動、ホイールで拡大縮小できます。
- 右側の勢力リストをクリックすると勢力詳細が見られます。
- 艦隊管理から艦船建造と武器/シールド/推進強化ができます。
