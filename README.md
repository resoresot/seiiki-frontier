# 星域フロンティア

スマホ向けのターン制宇宙ストラテジーです。

## v7 の主な変更

- 5つの恒星系をベースにした星域構造へ変更
- 各勢力は恒星系を持ち、その中の母星惑星を拠点に開始
- 最初は母星の衛星まで探索可能
- 研究「星系測量」で同一恒星系の別惑星探索を解放
- 研究「恒星ハーネス」「ワープ航法」で恒星チャージと星系間侵攻を解放
- 植民直後は維持費により収支が悪化し、開発で黒字化する設計へ変更
- 母星制圧で帝国敗退、残存拠点を接収
- 3ターン・9AP行動のチュートリアルに変更
- スマホで右メニューとマップが重なりにくい表示領域へ調整

## v9 fix
- タイトル画面では通常スクロール/ピンチを許可し、ゲーム画面だけマップ専用タッチ操作に切り替えます。
- 右側メニューとマップ操作ボタンが重ならないよう再配置しました。
- 2本指ピンチ後にドラッグ不能にならないよう、touchイベント専用処理へ切り替えました。


## Mobile v12 更新

- 研究数を大幅追加。共通研究と方針別研究を合わせ、分岐・合流・複数前提を増やしました。
- 研究画面を縦リストではなくツリーマップ表示に変更。取得済み経路は緑の線、研究可能な次ルートは青の線で表示します。
- 研究の一部は収入、植民コスト、開発コスト、艦隊戦力、恒星チャージ、AP上限に反映されます。


## mobile v13
- 研究ツリーを全体把握優先のコンパクトマップに変更。
- 研究項目はタップで詳細ポップアップを開き、前提・解放先・効果・現在資源を確認して実行。
- 各恒星系を5〜10惑星＋ランダム衛星に拡張し、衛星は親惑星にのみ接続。
- CPU難度を上げ、勢力性格と節目コメントを追加。


## mobile v17 changes

- CPU勢力が実際にプレイヤー領へ侵攻するようになりました。
- 母星を落とされると敗北画面が表示されます。
- 侵攻前に推定勝率・攻撃/防衛・防衛施設相性を確認できます。
- APを残したターン終了には確認を表示します。
- 開拓勅許と恒星市場の信用補助効果を実装しました。


## mobile v17 balance redesign

- 軍事・内政・技術の3勝利ルートをTurn50以降に揃える再設計を追加。
- 敵母星攻略に橋頭堡、包囲、首都継承令、降下軍団/要塞攻略教範、恒星チャージ2、戦力条件を追加。
- 支配勝利を単純な保有数ではなく、有効支配率・平均開発Lv・行政技術・赤字条件で判定。
- 技術勝利を最終技術だけでなく、研究拠点、電力/結晶収入、T4技術数で判定。
- CPUも内政/技術勝利を狙い、Turn50以降は母星攻略可能。
- 説明ページに勝利ルート進捗を表示。
