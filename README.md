# 星域フロンティア

ブラウザだけで動く、ターン制の宇宙ストラテジーゲームです。GitHub Pages にそのまま公開できます。

## v4 の主な改善

- 最初の2ターンを実操作ガイド化
- チュートリアル中に資源不足で止まらないよう補給処理を追加
- 上部の資源アイコンをクリックすると「何に使うか」「どう増えるか」を表示
- APバッジをクリックするとAPの意味を表示
- 星域マップをドラッグで上下左右に移動、ホイールで拡大縮小
- 右パネルの勢力をクリックすると、勢力詳細を表示
- 艦隊を「偵察艇 / フリゲート / 駆逐艦 / 空母」に分解
- 武器・シールド・推進強化を追加し、総合戦力・攻撃力・シールドに反映
- CPUターンの行動をカメラ移動つきで可視化

## 遊び方

1. 文明方針を選ぶ
2. 母星の隣を探索
3. 発見した星を植民
4. 開発で資源産出を増やす
5. 研究・艦隊建造・侵攻のどれを優先するか選ぶ

勝利条件は、支配勝利・技術勝利・覇権勝利の3種類です。

## 公開方法

このフォルダの中身を GitHub リポジトリ直下に置き、GitHub Pages を有効化してください。

- `index.html`
- `css/`
- `js/`
- `assets/`
- `.nojekyll`

## セーブ

セーブデータはブラウザの `localStorage` に保存されます。サーバーやデータベースは不要です。
