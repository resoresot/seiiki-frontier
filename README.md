# 星域フロンティア

スマホ向けのターン制宇宙ストラテジーです。

## v7 の主な変更

- 5つの恒星系をベースにした星域構造へ変更
- 各勢力は恒星系を持ち、その中の母星惑星を拠点に開始
- 最初は母星の衛星まで探索可能
- 研究「星系測量」で同一恒星系の別惑星探索を解放
- 研究「恒星ハーネス」「ワープ航法」で恒星チャージと星系間侵攻を解放
- 植民直後は維持費により収支が悪化し、開発で黒字化する設計へ変更
- 母星制圧で帝国敗退、残存拠点を接収
- 3ターン・9AP行動のチュートリアルに変更
- スマホで右メニューとマップが重なりにくい表示領域へ調整

## v9 fix

## Mobile v12 更新

- 研究数を大幅追加。共通研究と方針別研究を合わせ、分岐・合流・複数前提を増やしました。
- 研究画面を縦リストではなくツリーマップ表示に変更。取得済み経路は緑の線、研究可能な次ルートは青の線で表示します。
- 研究の一部は収入、植民コスト、開発コスト、艦隊戦力、恒星チャージ、AP上限に反映されます。


## mobile v13
- 研究ツリーを全体把握優先のコンパクトマップに変更。
- 研究項目はタップで詳細ポップアップを開き、前提・解放先・効果・現在資源を確認して実行。
- 各恒星系を5〜10惑星＋ランダム衛星に拡張し、衛星は親惑星にのみ接続。
- CPU難度を上げ、勢力性格と節目コメントを追加。
